# Udemy React Clone

This is a **React-based Udemy Clone** project deployed on **Vercel**. The project replicates the Udemy UI, including course categories, recommendations, a sale banner, and a navbar with interactive elements.

## 🚀 Live Demo
**[View Project](https://udemy-react-beryl-ten.vercel.app/)**

## 🚀 Features
✅ Dynamic Navbar – Includes search functionality and navigation links.
✅ Course Categories – Displays different learning categories.
✅ Trending Courses Section – Showcases the most popular courses.
✅ Personalized Course Recommendations – Displays recommended courses dynamically.
✅ Discount Offers & Sales Section – Highlights ongoing discounts.
✅ Fully Responsive Design – Works across desktops, tablets, and mobile devices.
✅ Footer with Useful Links – Contains links to important pages like support, privacy policy, etc.

## 🛠️ Installation & Setup

To set up the project locally, follow these steps:

### 1️⃣ Clone the Repository
```sh
 git clone https://github.com/thejayakumar-k/Udemy-React.git
 cd udemy-react-clone
```

### 2️⃣ Install Dependencies
```sh
 npm install
```

### 3️⃣ Start Development Server
```sh
 npm start
```
The application will be available at **http://localhost:3000/**.
